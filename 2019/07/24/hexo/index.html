개발 블로그

Hexo 빠르고 간단하고 강력한 블로그 프레임워크

블로그 제작을 도와주는 Nodejs 기반 블로그 프레임워크 Hexo를 소개합니다.

설치

사전준비사항 Github Pages 사용시

  1. github 계정생성
  2. git 설치
  3. 터미널 (terminal) 사용 ( 윈도우 : Git BASH 추천 | Mac : 터미널 ) 개인추천 | Mac : iTerm2
  4. 코드 에디터 (Code Editer) 개인추천 | Window || Mac VScode(Visual Studio Code)
  5. Window 명령어 , [Linux / Unix] 명령어 개인추천 | Git BASH 사용시 Linux / Unix 명령어 사용가능
  6. Node.js 설치
  7. npm 명령어 숙지
  8. 마크다운 문법 숙지 (Markdown | .md 파일)
  • cli 기반으로 진행된다. (command line interface)
  • 정상 실행결과
  • 브라우저에서 확인

http://localhost:4000
